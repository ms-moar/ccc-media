поподробнее - Claude API сам по себе не имеет "send to chat" — он только generate'ит text completion в HTTP response.                                                                                                                                    
                                                                                                                                                                                                                                                    
  Чтобы text попал в DM Dev / Safebot / Telegram / любой chat — обязательна отдельная application logic + transport API:                                                                                                                            
  - DM Dev → VEP /api/dm/.../agent-send
  - Safebot → /api/rooms/.../messages (signed payload)                                                                                                                                                                                              
  - Telegram → Bot API /sendMessage                               
  - etc.                                                                                                                                                                                                                                            
                                                                  
  Claude API = brain (text gen). Transport = mouth (где-то отправить). Они отдельные.                                                                                                                                                               
                                                                                                                                                                                                                                                    
  Если кто-то хочет "позвать Claude в DM" — нужна bridge: app слушает DM → когда видит mention → calls Claude API → posts result обратно в DM. Это и есть наш "claude-arch" pattern: я current Claude Code session с Bash-tool который делает curl  
  на DM API.
