# базовый лайфхак с агентами - настраиваете агента на каждую б

**Автор**: @pionerio
**Дата**: 07.03.2026 16:20
**Топик**: General

---

базовый лайфхак с агентами - настраиваете агента на каждую библиотеку/документацию приложения/датасет и они тригерятся, когда возникают проблемы и идут читать доки для поиска решения проблемы, гэпы могут быть на любом шаге, а 100% покрытия плана по моей статистике недостижимы, даже за 2-3 прохода с агентами практически никогда не получается достичь идеального исполнения.

PS

если это github репозиторий c документацией, лучше его склонировать в скоуп проекта + fallback через gh cli

оптимально если к доке есть mcp / cli

PS 2

все скиллы и агенты преимущественно пишу на англ, тк это нативный язык ИИ и он все равно будет переводить на него, поэтому будет меньше лаг на интерпретацию и качество лучше.
