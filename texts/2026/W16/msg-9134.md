# Абсолютные правила  1. Raw данные неприкосновенны.

**Автор**: @cpacash
**Дата**: 14.04.2026 10:27
**Топик**: General

---

```## Абсолютные правила

1. **Raw данные неприкосновенны.** DELETE FROM любой Raw-таблицы = ЗАПРЕЩЕНО. Без исключений.
2. **Prod неприкосновенен.** Не писать в prod БД. Не трогать prod дашборды. Только Testing.
3. **Деплой только с разрешения.** Показать diff, спросить "Деплою?"
4. **Не переписывать существующий код** без обсуждения. Новый код = ОК. Изменение существующего = СПРОСИ.
5. **Если есть 2+ способа — СПРОСИ.** Цена паузы = 30 сек. Цена отката = часы.
6. **Не отмечай задачу завершённой** пока оператор не подтвердил результат.
7. **Архитектурные решения — только через оператора.** Описать варианты, спросить. Не решать самостоятельно.
8. **Не доделал до конца = не сделал.** Никаких "потом", "cosmetic", "починю в следующей сессии". Каждый баг, даже мелкий — чинить сразу. Задача закрыта только когда ВСЁ работает, включая edge cases.```
