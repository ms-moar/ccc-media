🔌 ГАЙД: Плагины Claude Code — полный разбор

Многие не знают, но у Claude Code есть полноценная система плагинов. Вот всё что нужно знать: какие ставить, как юзать, и где подвохи.

═══════════════════════════════════

⚡️ ЧТО ТАКОЕ ПЛАГИНЫ

Плагины добавляют три вещи:
— Slash-команды (например /commit, /pr-review-toolkit:review-pr)
— Скиллы (Claude сам подтягивает когда нужно)
— Агенты (суб-агенты для специализированных задач)

Официальный маркетплейс (claude-plugins-official) уже подключён. Управление: /plugin → вкладки Discover / Installed.

═══════════════════════════════════

🏆 ЯДРО — СТАВЬ ОБЯЗАТЕЛЬНО

```
/plugin install context7
```
↳ Подтягивает актуальные доки прямо в контекст. Claude читает свежую документацию вместо устаревшей из памяти. 83k установок, самый популярный после frontend-design.

```
/plugin install security-guidance
```
↳ Хук безопасности. Предупреждает о command injection, XSS и небезопасных паттернах прямо при редактировании файлов. Anthropic Verified.

```
/plugin install commit-commands
```
↳ Git workflow одной командой: коммит, пуш, создание PR. Anthropic Verified.

```
/plugin install ralph-loop
```
↳ Итеративная доработка. Claude работает над задачей в цикле, видя предыдущие результаты, пока не доведёт до конца. Anthropic Verified.

```
/plugin install code-simplifier
```
↳ Рефакторинг и упрощение кода с сохранением функциональности. Anthropic Verified.

```
/plugin install code-review
```
↳ AI ревью PR с несколькими специализированными агентами. 57k установок. Anthropic Verified.

```
/plugin install feature-dev
```
↳ Полный цикл разработки фичи: исследование → дизайн → ревью. Anthropic Verified.

```
/plugin install frontend-design
```
↳ Продакшен-уровня фронтенд. Лидер по установкам — 120k. Anthropic Verified.
