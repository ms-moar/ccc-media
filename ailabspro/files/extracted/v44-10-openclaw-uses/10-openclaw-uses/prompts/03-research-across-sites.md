# Research Across Sites

## Cron Job Prompt

```
Create a cron job that runs daily at 6 AM to research the latest developments in AI, developer tools, and web technologies.

Sources to check:
- Hacker News front page
- GitHub Trending repositories
- Product Hunt new launches
- Major tech blogs and release pages for tools we use (Vercel, Next.js, React, Claude, OpenAI, etc.)

For each relevant item, provide:
- A brief summary of what it is
- Why it matters
- 2-3 potential video angles we could cover
- Source links for deeper reading

Organize the report by category (AI/ML, Developer Tools, Web Frameworks, Other) and post it to the #daily-research channel on Discord.
```
