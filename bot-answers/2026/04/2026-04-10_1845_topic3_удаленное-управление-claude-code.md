## Вопрос

удаленное управление claude code

## Ответ

Удалённое управление Claude Code — тема, которую группа обсуждала несколько раз. Вот что есть в базе знаний:

**1. Remote Control — нативная функция Claude Code**

Позволяет управлять локальной сессией с другого устройства через сайт claude.ai или мобильное приложение. Пишешь команды с телефона — они выполняются на компьютере в реальном времени. Соединение пробрасывается через сервера Anthropic, никаких туннелей или VPS настраивать не нужно.

Из обсуждения (03.03.2026): @indianajones_ok попробовал и рассказал что удобно — пишет с телефона, команды появляются в терминале на компе. @cpacash оценил что можно с мобилки без сомнительных костылей. @Papa_Misha считает что его TMX круче, но признал удобство для тех кого смущает бесконтрольность OpenClaw.

Важно: перед использованием нужно обновить CLI через `npm update -g @anthropic-ai/claude-code` — это выяснили @samaye10 и @mr_mojo_0 (дайджест за 16.03–23.03).

Ссылки:
https://code.claude.com/docs/ru/remote-control
https://t.me/c/3602915633/822/4804
https://raw.githubusercontent.com/ms-moar/ccc-media/main/texts/2026/W09/msg-4290.md

**2. Claude Dispatch агент**

Отдельный инструмент — Claude Cowork Dispatch — для синхронизации постоянного диалога между компьютером и телефоном с удалённым выполнением команд. @mr_mojo_0 поделился новостью (25.03.2026), @cpacash предупредил что скачивание приложения на телефон может привести к бану аккаунта.

Ссылки:
https://t.me/c/3602915633/822/6693
https://x.com/felixrieseberg/status/2034005731457044577?s=20

**3. ClaudeCodeUI — веб-интерфейс**

Открытый веб-интерфейс для управления сессиями и проектами Claude Code удалённо с мобильных или десктопа. @Papa_Misha скинул ссылку (18.02.2026). Работает также с Cursor CLI и Codex.

Ссылки:
https://t.me/c/3602915633/822/2860
https://github.com/siteboon/claudecodeui

**4. Управление через Telegram**

@denisso888 приводил ссылку на Claude Code channels — интеграцию управления сессией через Telegram.

Ссылка: https://github.com/anthropics/claude-plugins-official/blob/main/external_plugins/telegram/README.md

**5. Управление с Apple Watch**

@mr_mojo_0 поделился (дайджест за 23.03–30.03) опенсорсным инструментом для управления Claude Code с Apple Watch.

Ссылка: https://github.com/shobhit99/claude-watch

**6. Полноценный серверный сетап через tmux**

Если нужно работать 24/7 — @Papa_Misha рекомендует удалённый сервер с tmux. Его сетап: сервер за ~100$/мес (64 ГБ RAM + 192 ГБ swap), 100+ окон tmux, хуки обновляют статусы в названиях окон, 15 человек работают через веб-CRM. Для постоянных серверов рекомендует Contabo, для тестов — DigitalOcean, для CLI-управления серверами — Heroku или Scaleway.

Подробнее: https://t.me/c/3602915633/3/-10199
Полезнятина про tmux: https://t.me/c/3602915633/822/3845
