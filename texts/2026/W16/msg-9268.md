**🔥 Anthropic редизайнули Claude Code на десктопі і нафіг тепер курсор?**

Тепер можна запускати декілька сесій Claude одночасно в одному вікні, side by side, і керувати ними через новий сайдбар. Хто працював з Claude Code - розуміє, наскільки це було потрібно.

**Що ще додали:**
 • вбудований термінал (так, клод в терміналі, а тепер в термінал в клоді 😄)
 • редагування файлів прямо в інтерфейсі
 • превʼю HTML і PDF
 • швидший diff viewer
 • drag-and-drop лейаут, який можна налаштувати під себе.

І що важливо - всі CLI плагіни працюють так само, як і в командному рядку. Тобто нічого не зламали, просто зробили зручніше.

По суті Anthropic поступово перетворюють Claude Code з термінального інструменту на повноцінну IDE. І якщо чесно, з кожним таким оновленням все менше причин відкривати щось інше для роботи з кодом.
