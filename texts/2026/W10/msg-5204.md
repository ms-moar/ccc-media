VS Code 1.110 (February 2026) — Ключевое

Вышел **4 марта 2026** (1.110.1 с security fix). Главная тема — **агенты стали практичными**.

🔥 Топ фичи:

1. **Agent Plugins** — устанавливай пакеты skills/tools/hooks из Extensions
2. **Agentic Browser Tools** — агент сам управляет браузером, проверяет свои изменения
3. **Session Memory** — планы сохраняются между сообщениями
4. **Context Compaction** — ручная компактировка истории чата
5. **Fork a Chat Session** — клонируй чат для экспериментов
6. **Agent Debug Panel** — видишь tool calls, system prompts, hooks в реальном времени
7. **Background Agents** — отдаёшь задачу Copilot CLI, трекаешь в VS Code
8. **Claude Agents** — Claude Agent SDK через Copilot подписку (steering, queuing, compaction)
9. **Kitty Graphics** — рендер картинок прямо в терминале
10. **/yolo** — автоаппрув всех tool calls одной командой 😄
