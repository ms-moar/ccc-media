# У ПК есть один минус - большой расход токенов по сравнению с

**Автор**: @Papa_Misha
**Дата**: 07.04.2026 11:12
**Топик**: PaperClip

---

У ПК есть один минус - большой расход токенов по сравнению с крабом, потому что на каждую задачу будет грузиться весь обвес. Ну это как бы архитектура такая с этим ничего не сделаешь. Оптимизировать можно используя разные модели. Также сравнение с крабом по токенам не оч коректное потому что в ПК подписка а в крабе АПИ за монету теперь.
