baoyu-skills — 18 скиллов для Claude Code от китайского разработчика

  9.2k ⭐ | 1k форков | github.com/JimLiu/baoyu-skills

  Коллекция из 18 Claude Code скиллов в трёх группах. Самое интересное — CDP-паттерн
  для работы с X/Twitter через реальный Chrome.

  Визуальный контент (7 скиллов)

  — xhs-images: инфографика для Xiaohongshu, 9 стилей × 6 layouts
  — infographic: 20 типов layout (pyramid, fishbone, venn) × 17 стилей
  — cover-image: обложки статей, 5D система (Type × Palette × Rendering × Text × Mood)
  — slide-deck: статья → слайды → .pptx + .pdf, 16 пресетов
  — comic: образовательные комиксы с панелями
  — article-illustrator: авто-иллюстрации по структуре статьи
  — post-to-x: публикация в X через Chrome CDP, обходит anti-bot

  AI-бэкенды (2)

  — image-gen: мульти-провайдер (OpenAI, Google, DashScope, Replicate и др.)
  — danger-gemini-web: reverse-engineered Gemini Web API

  Утилиты (6)

  — url-to-markdown: Chrome CDP → Defuddle → clean markdown + HTML snapshot
  — danger-x-to-markdown: X → markdown через reverse-engineered API (threads +
  Articles)
  — compress-image, format-markdown, markdown-to-html, translate (3 режима качества)

  Что ценного

  url-to-markdown — production-validated паттерн: рендерит JS через Chrome CDP, парсит
  через Defuddle. Ровно то, что мы проектируем для extraction pipeline.

  danger-x-to-markdown — альтернатива FxTwitter API для тредов и X Articles. Prefix
  danger- = reverse-engineered, может сломаться.

  Инфографика впечатляет: 20 layouts × 17 стилей = 340 комбинаций с превью каждой.
