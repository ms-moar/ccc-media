# AutoMaker — это десктопная AI-студия разработки. По сути

**Автор**: @pavel2077
**Дата**: 24.02.2026 11:53
**Топик**: Claude Code

---

**AutoMaker** — это десктопная AI-студия разработки. По сути Kanban-доска, где ты описываешь фичи, а AI-агенты (на базе Claude Agent SDK) сами их кодят.

Как работает:

1. Добавляешь фичу на Kanban (текст, скриншоты, картинки)
2. Перетаскиваешь в "In Progress" → агент автоматически берёт задачу
3. Агент пишет код, запускает тесты, делает коммиты — всё в изолированном git worktree
4. Ты ревьюишь, аппрувишь, мержишь

Стек:

• React + Vite + Electron (или браузер)
• Express бэкенд
• Claude Agent SDK под капотом
• Git worktree isolation (каждый агент в своей ветке)

По сути:

Это **визуальная обёртка над Claude Code CLI** с project management слоем. Типа Cursor, но вместо "ты пишешь код с AI" — "ты описываешь что надо, AI делает всё сам".
