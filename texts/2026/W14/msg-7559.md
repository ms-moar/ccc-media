🔥 **KAIROS — автономний агент 24/7** — вже побудований, за feature flags. Включає:

Тобто вони будують те саме що ми з Brain Suite — persistent memory + always-on agent. Ми паралельно йдемо.

🛡 **Anti-distillation** — сервер інжектить фейкові tools в system prompt, щоб отруїти тренувальні дані якщо хтось записує API трафік. Плюс connector-text summarization з криптопідписом.

🧠 **3-рівнева компактація контексту:**

• MicroCompact — локальна чистка на льоту
• AutoCompact — біля ліміту, 13K→20K summary з circuit breaker
• Full Compact — emergency, 50K бюджет, selective re-injection

В нас аналог — hippocampus decay + defrag, але вони більш гранулярні.

🤖 **Coordinator mode** — один Claude керує кількома worker Claude з обмеженими toolsets. Оркестрація через промпт, не код. Правила типу "Do not rubber-stamp weak work".

😂 **Frustration detection через regex** — `wtf|shit|fuck|horrible` — AI компанія юзає regex для sentiment analysis. Дешево і сердито.

🔐 **Native attestation (Bun/Zig)** — DRM для API calls. Hash рахується нижче JS runtime в Zig, сервер валідує що це реальний Claude Code бінарник. Через це OpenCode не може юзати Claude API напряму.

🐛 **Undercover mode** — ховає що AI писав код. Немає force-OFF. Тобто PR від антропіківців в open source не показують що це AI.

**Для нашого Brain Suite релевантне:**

1. Їхній `/dream` = наш defrag. Ми попереду — у нас вже працює.

Цей лік підтверджує що наша архітектура (persistent memory + background processing + multi-agent) — правильний напрямок. Антропік будує те саме, просто з іншого боку.
