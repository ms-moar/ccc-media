# CCC Media Repo

Медиа-хранилище группы **Claude Code Club** — изображения, видео, тексты и ресурсы AI Labs Pro.

Используется ботом и дайджест-системой для хостинга файлов со ссылками через `raw.githubusercontent.com`.

## Структура

```
ccc-media/
├── 2026/                      # Медиа из группы (по ISO-неделям)
│   ├── W06/                   # Неделя 06 (3 февраля)
│   ├── W07/                   # Неделя 07 (10 февраля)
│   ├── W08/                   # Неделя 08 (17 февраля)  — 98 файлов
│   └── W09/                   # Неделя 09 (24 февраля)  — 41 файл
│
├── texts/                     # Полные тексты длинных сообщений
│   └── 2026/W{NN}/msg-{id}.md
│
└── ailabspro/                 # Материалы курса AI Labs Pro
    ├── files/                 # Исходные архивы + распакованные файлы
    │   ├── v5-rules.txt       # Версионированные ресурсы (v5–v45)
    │   ├── v11-docs.zip
    │   └── extracted/         # Распакованные архивы (33 папки)
    │       ├── v11-docs/
    │       ├── v23-figma-agent/

        ├── v43-post.png

```

## 2026/ — Медиа группы

Фото, скриншоты, видео и документы из чата, организованные по ISO-неделям.

- **Именование**: `msg_{id}.jpg`, `msg_{id}.mp4`, `msg_{id}_sticker.webp`
- **Типы**: скриншоты кода/настроек, мемы, схемы, видео-демо
- **Всего**: ~120 изображений, ~13 видео

Сохраняются автоматически скриптом `tg_archive_messages.py` при архивации сообщений группы.

## texts/ — Длинные тексты

Полные тексты объёмных сообщений (промпты, конфиги, CLAUDE.md, SOUL.md), которые не помещаются в дайджест.

- **Формат**: markdown с метаданными (автор, дата, топик)
- **Именование**: `msg-{id}.md`
- **Всего**: 25 файлов

Сохраняются автоматически при генерации полезнятины. Ссылки вставляются в посты дайджеста.

## ailabspro/ — Ресурсы курса

Материалы AI Labs Pro: промпты, CLAUDE.md файлы, skills, guides, workflows, шаблоны проектов.

- **files/**: ZIP-архивы и распакованные ресурсы версий v5–v45
- **posts/**: тексты постов с описанием ресурсов
- **screenshots/**: скриншоты постов курса
- **Всего**: ~350 файлов, 24 MB

Проиндексированы в ChromaDB для поиска через бота группы (`@pm_claude_bot`).

## Использование

### Raw URL для файла

```
https://raw.githubusercontent.com/ms-moar/ccc-media/main/2026/W08/msg_2298.jpg
```

### В дайджестах и полезнятине

Скрипт `github_media.py` загружает файлы и возвращает raw-ссылки для вставки в посты Telegram.
