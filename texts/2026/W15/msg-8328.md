# Ограничения:   - Оба прокси НЕ поддерживают кастомные tools

**Автор**: sklip
**Дата**: 06.04.2026 15:22
**Топик**: OpenClaw

---

Ограничения:
  - Оба прокси НЕ поддерживают кастомные tools (sessions_send и т.д.)
  - Node прокси использует --print (one-shot), без tool_use
  - Rust прокси поддерживает Anthropic формат, но Claude CLI игнорирует кастомные tools из запроса
  - От root не работает (--dangerously-skip-permissions запрещён под root)
