**⚠️ macOS атакуют через фейковый Claude**

Пользователи macOS при попытке установить Claude Code могут наткнуться на фейковую страницу в поиске, после чего [установят](https://www.prnewswire.com/news-releases/ai-empowered-bybit-security-team-uncovers-macos-malware-campaign-targeting-users-searching-for-claude-code-302748925.html) себе вирусное ПО под видом ИИ-помощника.

Вирус нацелен на:
▪️ Учетные данные
▪️ Криптокошельки
▪️ Telegram-сессии
▪️ Удаленный доступ к устройству

Вирус обходит защиту macOS и прописывается в автозагрузке, чтобы запускаться после нового входа в систему.

__Несмотря на критичные последствия заражения, сама атака имеет максимально простую стартовую точку – просто не качайте ничего по ссылкам из поиска 😁__
