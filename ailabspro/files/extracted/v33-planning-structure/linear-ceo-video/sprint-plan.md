# Sprint Planning Document

## Sprint Overview
Create a high-level sprint timeline based on the feature roadmap from the PRD. Ask the user for sprint duration preference (1-2 weeks) and project deadline.

## Sprint 1
Break down Phase 1 features into specific tasks with story point estimates. Confirm task priorities with the user.

### Sprint Backlog
Generate user stories and tasks from the PRD's first milestone. Include acceptance criteria for each item.

### Definition of Done
Define completion criteria based on the project's technical requirements and quality standards from the PRD.

### Risks & Blockers
Identify potential obstacles from technical dependencies and propose mitigations. Ask the user about resource constraints.

## Sprint 2
Plan Phase 2 features based on Sprint 1 deliverables. Adjust scope based on initial velocity assumptions.

### Sprint Backlog
Generate user stories for the next feature set. Reference dependencies from Sprint 1.

### Definition of Done
Refine completion criteria based on any new technical decisions or user feedback.

### Technical Decisions
Document architecture and implementation choices that affect future sprints. Derive from technical spec if available.

## Sprint 3-6
Extend the sprint structure for remaining phases. Scale estimates based on projected velocity.

## Velocity Tracking
Create a tracking table for planned vs completed story points. Update projections as sprints complete.

## Ceremonies Schedule
Propose a schedule for standups, reviews, and retrospectives. Confirm timing preferences with the user.

## Team Roles
Define responsibilities based on project scope. Ask the user about team composition and availability.

## Definition of Ready
Establish criteria stories must meet before sprint inclusion. Base on project complexity and dependencies.

## Definition of Done
Set sprint-level completion standards aligned with the PRD's quality requirements.

## Backlog Grooming Notes
Maintain a prioritized list of upcoming work derived from the full feature roadmap.

### Upcoming Stories
List stories for future sprints based on PRD phases not yet scheduled.
