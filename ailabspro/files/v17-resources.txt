Initial Prompt: I want to create an AI note-taking application that takes manual notes and then uses AI to generate summaries of those notes. An additional feature would be voice note-taking capability. Help me brainstorm this idea.
Backend Prompt: Build a Node.js/Express REST API backend for an AI note-taking app with MongoDB database. Implement JWT authentication, file upload to cloud storage (S3/Google Cloud), and integrate OpenAI/Claude API for text summarization and Google Speech-to-Text for voice transcription. Create endpoints for notes CRUD, document processing, voice recording management, global search, and folder/tag organization. Include background job processing for AI tasks, WebSocket real-time updates, proper error handling, and secure user data isolation with encrypted storage.
Frontend Prompt: Develop a modern AI note-taking web app using Next.js 14+, TypeScript, and Tailwind CSS. Build three core features: a rich text editor for manual notes, drag-and-drop document upload with viewer, and voice recording with transcription display. Include user authentication, responsive dashboard with sidebar navigation, global search across all content, and folder/tag organization system. Implement auto-save, real-time processing status, and mobile-first responsive design. Focus on clean UI, smooth animations, and excellent user experience with proper error handling and loading states.
Auth Prompt: Build a complete authentication system with Node.js/Express backend and Next.js frontend. Backend: Create JWT-based auth with MongoDB, bcrypt password hashing, user registration/login endpoints, protected middleware, and optional OAuth integration. Frontend: Build responsive login/register forms with React Context state management, protected routes with automatic redirects, token storage, and proper loading/error states. Implement password reset flow, profile management, and ensure seamless integration between frontend auth context and backend JWT validation with proper security practices.
