🔒 БЕЗОПАСНОСТЬ — ВАЖНО

Официальный маркетплейс: Anthropic проводит базовый автоматический ревью. Плагины с бейджем «Anthropic Verified» проходят дополнительную проверку.

НО: Anthropic не контролирует какие MCP серверы и софт включены в плагины. Community-плагины могут устанавливать непроверенное ПО.

Правила:
✅ Из официального маркетплейса с Verified — ставь спокойно
✅ Перед установкой смотри исходники на GitHub
✅ Используй --scope project для изоляции
⚠️ Сторонние маркетплейсы — как любой open source: смотри звёзды, автора, код

⚙️ PLAYWRIGHT НЕ РАБОТАЕТ? ЧАСТАЯ ПРОБЛЕМА

Если поставил playwright плагин, перезагрузил, а Claude говорит что у него нет таких инструментов:

1. Установи браузеры (выйди из Claude Code → в терминале):
```
npx playwright install
```

2. Зайди обратно и проверь:
```
/mcp
```
↳ Найди playwright в списке — должен быть connected.

3. В первый раз явно скажи:
«Используй playwright MCP — открой https://mysite.com и сделай скриншот»
Иначе Claude может попытаться запустить Playwright через Bash вместо MCP.

4. На Windows: закрой Chrome перед использованием — Playwright не может запустить второй экземпляр, если Chrome уже открыт.
