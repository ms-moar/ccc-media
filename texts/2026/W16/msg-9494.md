# Это рекомендуемый фреймворк настройки скиллов и агентов - че

**Автор**: @pionerio
**Дата**: 17.04.2026 15:25
**Топик**: Claude Code

---

Это рекомендуемый фреймворк настройки скиллов и агентов - чекпоинты и гейты. Чекпоинты нужны для отката на предыдущий шаг, гейты, чтобы не проходить важные части реализации без доп проверок. 

Пример. У тебя описан дев скилл по TDD, но не зафиксированы гейты, лбюбой шаг может быть произвольно пропущен по решению агента, если нет гейтов.
