**
Paperclip** — open-source оркестратор для "компаний без людей". По сути — надстройка над AI-агентами (включая OpenClaw, Claude, Codex, Cursor) которая добавляет:

• **Org chart** — иерархия агентов с ролями (CEO, CTO, CMO...)
• **Goal alignment** — каждая задача привязана к миссии компании
• **Ticket system** — общение через тикеты, полный audit log
• **Бюджеты** — месячный лимит на агента, не превысит
• **Multi-company** — несколько бизнесов в одном деплое
• **Heartbeats** — агенты просыпаются по расписанию

**Для нас:**

Честно — мы уже делаем 80% этого руками через OpenClaw:

• Org chart → наш `IDENTITY.md`
• Goal alignment → `MEMORY.md` + active-tasks
• Heartbeats → наши heartbeats
• Multi-company → OFMStars + Balance 1

**Что Paperclip добавляет сверху:**

• Визуальный дашборд с org chart и бюджетами (у нас пока нет)
• Тикет-система с полным трейсингом (у нас Git + PR)
• Единый cost tracking по агентам (у нас вручную)

**Моё мнение:** интересно посмотреть, но менять стек сейчас — риск. Мы уже настроены и работаем. Стоит попробовать на изолированном side project, не на боевых бизнесах. Плюс он ранний — open-source, но непонятно насколько стабильный.
