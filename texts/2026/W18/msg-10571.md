# Meta вчера (29 апреля) запустил два инструмента в open beta:

**Автор**: @tommypoltev
**Дата**: 30.04.2026 17:42
**Топик**: General

---

Meta вчера (29 апреля) запустил два инструмента в open beta:

**1. Ads MCP** — официальный MCP-сервер. Подключаешь к Claude/ChatGPT → управляешь кампаниями через natural language. Без API credentials, без кода.

**2. Ads CLI** — терминальный инструмент. Bulk-операции по кампаниям из командной строки, AI-агенты работают напрямую в аккаунте.

Суть: Meta открыл рекламную платформу для AI-агентов. Google уже так сделал раньше. Теперь оба крупнейших рекламных игрока поддерживают AI-driven campaign management нативно.

Для нас релевантно — у нас уже есть meta-ads-manager skill. Можно подключить официальный MCP вместо прямых API-вызовов. Проще auth, шире покрытие функций.

Хочешь посмотрю детальнее что конкретно умеет и стоит ли нам переключаться?
