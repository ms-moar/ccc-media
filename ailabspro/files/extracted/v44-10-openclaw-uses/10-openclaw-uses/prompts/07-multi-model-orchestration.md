# Multi Model Orchestration

## Chat Prompt

```
Build a landing page for [describe your app/product] in the directory ~/projects/my-landing-page.

Use Claude Code to implement the website code. Use the Nano Banana Pro skill to generate images that fit the application's theme and style. Save all generated images to the public directory so Claude Code can reference them in the page.

Break the work into steps:
1. Plan the landing page layout and identify what images are needed
2. Generate the images using Nano Banana Pro
3. Implement the page using Claude Code with the generated images
4. Commit to Git after each major step

Run Claude Code with the dangerously-skip-permissions flag to avoid permission prompt timeouts.
```
