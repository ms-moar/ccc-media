Clawd Bot — что сделано

  Суть: Заменили OpenClaw на свой Telegram бот на Claude Agent SDK. Причина — Anthropic заблокировал OAuth токены для third-party harnesses (OpenClaw).

  Архитектура:
  Telegram → grammy (Node.js) → Claude Agent SDK → Claude Max подписка
                                      ↓
                                31 агент, каждый со своим:
                                - SOUL.md (инструкции)
                                - модель (haiku/sonnet)
                                - tools (Bash, WebSearch, Read...)
                                - память (markdown файлы)

  Что работает:
  - 2 бота в одном процессе (main + barsik)
  - Роутинг по keywords из config.json — мгновенный, без токенов
  - User bindings — персональные агенты по userId
  - Голосовые → Whisper (Groq) → текст → агент
  - Email watcher — IMAP IDLE, спам-фильтр, AI классификация
  - Память — markdown файлы + SQLite сессии + vault
  - MCP серверы — pinchtab, camoufox
  - Cron — экономический календарь пн-пт 09:00
  - Скрипты агентов (pass24, weather, travel...)

  Стек:
  - Node.js + grammy + @anthropic-ai/claude-agent-sdk
  - Python (email watcher, скрипты)
  - SQLite (сессии, логи)
  - Markdown (память, vault)
  - Claude Max подписка (без API ключей)
