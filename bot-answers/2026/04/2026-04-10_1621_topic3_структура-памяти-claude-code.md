## Вопрос

структура памяти claude code

## Ответ

**Структура памяти Claude Code**

В группе тема памяти обсуждалась активно. Вот основные подходы и решения:

**Встроенная память КК**
По данным @Papa_Misha (05.02), для постоянной памяти используется `CLAUDE.md` — туда записывается всё важное, что нужно запомнить навсегда. Контекст хранится в папке `.claude/projects`.

По данным @dimaumbo и @Papa_Misha (дайджест за 09.02–16.02), при переполнении контекста работает команда `/compact` с autocompact buffer на 33k токенов. Claude Code удаляет историю чатов через 30 дней по умолчанию, но это можно изменить через `cleanupPeriodDays` в `~/.claude/settings.json`.

**Проблема потери контекста**
@Papa_Misha (05.02) признал: "глобально вопрос контекста нормально так и не решён — он один хер забывает периодически". Но постоянное документирование проекта значительно помогает.

**Что класть в CLAUDE.md**
По рекомендациям @Papa_Misha (дайджест от 13.02):
— Полное описание проекта (архитектура, структура файлов, схема БД, ключевые flow, конфигурация)
— Промпт для поведения после compact: вспомнить задачу, перечислить сделанное, определить следующий шаг
— Хук, который в конце задачи проверяет, не нужно ли обновить документацию

**SOUL.md**
По данным @tommypoltev (13.02, дайджест памяти агентов), SOUL.md — это файл с ролью агента. Читается при старте сессии, так инструкции не забываются при длинных сессиях. @tommypoltev реализовал у каждого агента свою папку с AGENTS.md, MEMORY.md, SOUL.md, USER.md и daily logs в `memory/YYYY-MM-DD.md`.

**Инструкции агента**
По данным из дайджеста за 09.02–16.02, инструкции лучше фиксировать через hooks с `promptUserSubmit` или в SOUL.md. @cpacash жаловался что КК забывает инструкции после 5-6 задач — именно поэтому важно прописывать жёсткие инструкции в agent.md.

**Внешние инструменты памяти**
— **claude-mem** — плагин от @mestniy_d (25.03), работает как архив памяти, автоматически сохраняет решения и контекст. Ссылка: https://claude-mem.ai
— **claude-note** — опенсорс-утилита от @Papa_Misha (13.02), слушает сессии через хуки и раскладывает знания в Obsidian
— **NotebookLM** — @Papa_Misha (18.02) поделился видео-интеграцией как внешний memory банк: https://m.youtube.com/watch?v=eFCHwtufjJc
— **ByteRover** — облачная память с алгоритмом консолидации, бесплатная версия тоже бустит контекст. По данным @Papa_Misha (13.02)

**Проверка при странном поведении**
По совету @pionerio (02.04, полезнятина): после обновлений модели возможны проблемы с кэшем и memory-файлами — туда могли записаться старые команды. Проверяйте кэш и мемори при странном поведении.
Ссылка: https://t.me/c/3602915633/822/7707

**1 млн токенов**
По данным @indianajones_ok (10.03), по подписке появился 1 млн токенов контекста — контекст забивается медленнее, автокомпакт срабатывает реже. После автокомпактов на 200к модель "просыпается дезориентированной".
Ссылка: https://t.me/c/3602915633/822/5465
